# 구글 클라우드 콘솔 가입하기

<img src="assets/images/memorit/google/google-personal.png" />

개인 개발자 계정으로 생성하시려면 '개인' 으로 시작하기를 눌러주세요.

해당 글에서는 개인 개발자 생성 기준으로 글이 작성됩니다.

개인개발자 계정으로 생성하는 것이 아니라면 D-U-N-S 번호가 필요합니다.
D-U-N-S 번호는 아래의 링크에서 발급받는 방법이 자세히 나와있습니다.
https://support.cafe24.com/hc/ko/articles/26939754488345-D-U-N-S%EB%B2%88%ED%98%B8%EB%8A%94-%EC%96%B4%EB%96%BB%EA%B2%8C-%EB%A7%8C%EB%93%9C%EB%82%98%EC%9A%94

<img src="assets/images/memorit/google/google-2.png" />

<img src="assets/images/memorit/google/google-3.png" />

개발자 이름을 입력해주고, 다음을 눌러주세요.

<img src="assets/images/memorit/google/google-4.png" />

결제 프로필 만들기 또는 선택 버튼을 눌러 프로필을 생성해주세요.

프로필이 생성된 후 내 정보, 앱 정보, google에서 연락하는 방법을 작성해주세요.

<img src="assets/images/memorit/google/google-5.png" />

개인 개발자 계정이 생성이 완료되었다면 사진이 포함된 페이지로 이동됩니다.

여기서 본인 확인을 해주어야합니다. 본인 인증이 되지않으면 아무것도 할 수 없습니다.

90일 이내로 발급받은 문서를 제출하라하는데, 관련 문서가 없다면 정부24에서 주민등록등본을 발급받아서 제출하면 됩니다.
