    Giscus 블로그 댓글 기능 사용하기

    1. 댓글을 관리할 github repository 를 하나 만들어줍니다.

    저는 GiscusComment 라는 이름으로 만들었습니다.

    1. 만든 Repo 에 들어가서 General => Features => Discussions 를 활성화 해줍니다.

    2. 그러면 Discussions 탭이 추가가 됩니다. 들어가서 왼쪽 Categories 에서 연필 모양을 눌러주고 New category 를 눌러줍니다.

    3. Category name: Comment, Discussion Format: Announcement 로 설정해주고 생성해줍니다.

    4. Giscus앱 추가 를 해주겠습니다. GiscusComment 레포지토리만 선택해서 추가해주겠습니다.

    5. Giscus 설정 페이지에 들어가줍니다.

    6. 저장소에 username/comment-repo 를 입력해주고, Discussion 카테고리는 Comment로 바꾸어줍니다.

    7. 저는 다른 것은 바꾸지 않았지만 원한다면 바꾸셔도 됩니다.

    8. script 부분을 복사하여 웹 사이트 템플릿에 붙여넣어 줍니다.

    9. 저처럼 vuepress home theme 를 이용하시는분은 다음과 같이 해주시면 됩니다.

    AWS S3 배포하기 (Vue3 + Quasar + pnpm)

    AWS S3 배포하기 (Vue3 + Quasar + pnpm)

    확장성이 뛰어나고 경제적인 클라우드 스토리지 서비스인 AWS(Amazon Web Services) Simple Storage Service(S3) 로 배포를 진행하겠습니다.

    프로젝트를 AWS S3에 구축하면 AWS 클라우드의 확장성, 안정성 및 비용 효율성을 활용할 수 있습니다.

    또한 AWS S3는 정적 웹 사이트를 쉽게 제공하여 배포 프로세스를 단순화하고 인프라의 복잡성을 줄일 수 있습니다.

    AWS 계정이 없다면 만들어주세요.
    github actions를 이용할 것이니 github repository에 프로젝트를 올려주세요.
